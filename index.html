// 01

Физические явления

Механика
Гармонические колебания

Периодическое движение, при котором тело возвращается в положение равновесия под действием силы, пропорциональной смещению. Основа анализа вибраций и волновых процессов.

F = −k·x  ·  T = 2π√(m/k)
Термодинамика
Теплопроводность

Перенос теплоты от нагретых тел к холодным путём непосредственного взаимодействия молекул. Интенсивность определяется коэффициентом теплопроводности материала.

q = −λ·(dT/dx)  [Вт/м²]
Электромагнетизм
Электромагнитная индукция

Возникновение ЭДС в замкнутом контуре при изменении пронизывающего его магнитного потока. Фундамент работы генераторов, трансформаторов и электродвигателей.

ε = −dΦ/dt    Φ = B·A·cos α
Оптика
Преломление света

Изменение направления распространения световой волны на границе двух сред с различными оптическими плотностями. Описывается законом Снеллиуса через показатели преломления.

n₁ sin θ₁ = n₂ sin θ₂
Квантовая физика
Волновая функция

Комплексная функция, описывающая квантовое состояние системы. Квадрат её модуля |ψ|² задаёт плотность вероятности обнаружить частицу в данной точке пространства.

iℏ ∂ψ/∂t = Ĥψ  (уравнение Шрёдингера)
Акустика
Стоячие волны

Интерференционная картина, возникающая при наложении двух бегущих волн одинаковой частоты и амплитуды, распространяющихся навстречу. Лежит в основе работы музыкальных инструментов.

λₙ = 2L/n  ·  fₙ = n·v/(2L)

Механизмы

// зубчатая передача
Передаточное отношение i = z₂/z₁ 2.00

i = z₂/z₁ = ω₁/ω₂. При i>1 — понижающая передача: скорость падает, крутящий момент растёт пропорционально.

// рычаг первого рода
Положение точки опоры 50%

F₁·d₁ = F₂·d₂ — золотое правило механики. Чем ближе опора к грузу — тем меньше нужна приложенная сила, но тем больше путь.

// кривошипно-ползунный механизм

Преобразует вращение кривошипа в возвратно-поступательное движение ползуна. Основа двигателей внутреннего сгорания, компрессоров и насосов.

// полиспаст (система блоков)
Число подвижных блоков n 2

Выигрыш в силе = 2n раз (n — число подвижных блоков). КПД снижается с увеличением числа блоков из-за трения.
